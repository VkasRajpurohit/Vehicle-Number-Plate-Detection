About the Vehicle Number Plate Detection

Vehicle Number Plate Detection

Built a deep learning based model for Vehicle Number Plate Detection System that would be able to get an input as vehicle image and provide the result as extracted number from vehicle number plate.

In the Project, It is shown that how to process image data, detect objects inside the image and create bounding box for the detected object. Inception-ResNet-v2 transfer learning Tensorflow model used for this project, OpenCV for this project in order to identify the license number plates and the python PyTesseract for the characters and digits extraction from the plate. {% endblock body %}
